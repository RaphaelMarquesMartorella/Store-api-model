## Run Locally

Clone the project

```bash
  git clone https://github.com/RaphaelMarquesMartorella/beauty-store-api.git
```

Go to the project directory

```bash
  cd /
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
